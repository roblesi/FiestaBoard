# Images Directory

This directory contains screenshots and images used in the main README.md file to showcase the Vestaboard Display Service.

## Purpose

Images in this folder are displayed in the main README to:
- Show the Vestaboard displays in action
- Demonstrate the web UI interface
- Provide visual examples of different features
- Make the project more appealing to potential users

## Naming Conventions

Use descriptive, lowercase filenames with hyphens:
- `vestaboard-hero.png` - Main hero image at top of README
- `web-ui-home.png` - Web UI screenshot
- `weather-display.png` - Weather + DateTime display example
- `home-assistant-display.png` - Home Assistant status display
- `star-trek-quotes-display.png` - Star Trek quotes display
- `guest-wifi-display.png` - Guest WiFi credentials display
- `muni-display.png` - Muni transit display
- `traffic-display.png` - Traffic conditions display
- `stocks-display.png` - Stock prices display
- `surf-display.png` - Surf conditions display
- `air-fog-display.png` - Air quality and fog display

## Image Guidelines

- **Format**: PNG recommended for screenshots (better quality for text)
- **Size**: Keep file sizes reasonable (<500KB when possible)
- **Dimensions**: Use consistent aspect ratios for similar image types
- **Optimization**: Compress images before committing to reduce repository size

## Adding New Images

1. Take or create the screenshot/image
2. Optimize the image (compress if needed)
3. Save with a descriptive filename following the naming convention
4. Update README.md to reference the new image using relative path: `![Description](./images/filename.png)`
